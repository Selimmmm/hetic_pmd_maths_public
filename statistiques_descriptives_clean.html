I. Notions préliminaires¶

a. Vocabulaire¶

  • Population
    • Ensemble des individus ou des objets d'intrt
  • Echantillon
    • Seulement une partie de la population d'intrt
  • Echantillon alatoire
    • Seulement une partie de la population d'intrt, dont les individus ont t choisis au hasard
  • Echantillon reprsentatif
    • Seulement une partie de la population d'intrt, qui la reprsente fidlement
  • Paramtre
    • Une valeur qui concerne une population
  • Statistique
    • Une valeur qui concerne un chantillon

b. Application de ces concepts¶

  • Dans l'immense majorit des cas, on travaille avec les donnes d'un chantillon et pas de la population complte
  • Dans ce cas, on travaille par "infrence statistique" : on induit des caractristiques sur un groupe gnral (la population), partir d'informaton sur un groupe particulier (l'chantillon). On s'intresse galement l'erreur que l'on peut commettre entre la vraie valeur (le paramtre de l'chantillon) et la valeur obtenue partir de l'chantillon (la statistique)
  • Attention, un chantillon alatoire, n'est pas toujours reprsentatif

II. Types de variables et représentations¶

a. Types de variables¶

  • Variable catgorielle ou qualitative
    • Une variable qui est une catgorie : la couleur des yeux pour un humain, la marque pour une voiture...
    • Les diffrentes valeurs possibles sont appeles les modalits
      • Nominale :
        • Ne dpend pas d'un ordre, par exemple la couleur des yeux
      • Ordinale :
        • Les valeurs peuvent tre ordonnes, par exemple : mauvais / moyen / bien / trs bien
  • Variable numrique ou quantitative
    • Une variable qui est une valeur, un nombre. Peut tre continue ou discrte
      • Variable discrte
        • On peut numrer toutes les valeurs possibles : le nombre de personnes dans une salle, le nombre d'annes d'tudes
      • Variable continue
        • Peut prendre n'importe quelle valeur entre deux bornes : la taille d'un humain, la vitesse du vent

values = diamonds['cut'].value_counts()
labels = np.arange(len(diamonds['cut'].unique()))

yvalues = []
xvalues = range(len(diamonds['cut'].unique()))

tmp = 0
for x in diamonds['cut'].value_counts():
    tmp += x/len(diamonds)
    yvalues.append(tmp)
    
    
fig, ax1 = plt.subplots()
ax2 = ax1.twinx()
    
ax1.bar(labels, values, align = 'center')
ax1.set_ylabel('Effectif')
ax2.plot(xvalues, yvalues, color = 'orange', linewidth = 2, linestyle = '-')
ax2.set_ylim(0,1)
ax2.set_ylabel('Frquence cumule')
plt.xticks(labels, diamonds['cut'].unique())
plt.title('Variable Cut Diagramme de Pareto')

plt.show()
